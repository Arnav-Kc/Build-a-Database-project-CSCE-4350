# Build-a-Database Key-Value Store
EUID: your_euid_here (ak1334 in my case)

A simple persistent key-value store using append-only storage.

## The command to run the file
python kvstore.py

## The various commands to get various outputs
SET <key> <value> - A value is stored
GET <key> - A value is retrieved
EXIT - You're exited out of the program

## Storage
All data is persisted to data.db using an append-only log.
